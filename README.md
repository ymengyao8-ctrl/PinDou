# 拼豆图纸生成器

一个纯前端的拼豆图纸生成工具。用户上传图片后，可以按常见拼豆板尺寸生成带色号、坐标和用量统计的图纸。

## 功能

- 上传图片并生成拼豆图纸
- 支持 MARD、Artkal S 色号体系
- 支持 29x29、50x50、72x72、100x100 等常见板面尺寸
- 可选择完整显示或铺满裁切
- 支持实用大色块、超简大色块、均衡、接近原图等精细度
- 支持只显示某一种色号
- 支持将一个色号替换为另一个相近色号
- 支持按色号或用量排序统计
- 支持主体框选抠图，框外自动留空
- 支持导出 PNG 图纸

## 本地运行

```bash
python3 -m http.server 8123 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8123/
```

## 部署

这是一个静态站点，直接用 GitHub Pages、Netlify、Vercel 或任意静态文件服务器托管即可。
